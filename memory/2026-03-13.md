## 🔧 技术笔记

### OpenClaw 限制
1. 不支持自定义钩子配置（hooks.on_message_received 等）
2. plugins 配置需要对象格式，不是数组
3. 子代理 thread=true 需要渠道插件支持

### 浏览器安全策略
- `file://` 协议无法 fetch 本地 JSON 文件
- 解决方案：`python3 -m http.server 8000`

## 📝 文章创作流程规范（与 jing 合作）

### 核心流程（强制）⭐

**阶段零：信息收集与思考**
1. 阅读用户提供的原始信息
2. 提取核心思想、关键词、情感倾向
3. 思考文章独特价值、用户真正想表达的
4. 整理后复述理解的核心思想

**阶段一：需求确认（7 个问题）**
1. 主题/核心思想
2. 文章基调（哲学思辨 / 行业洞察 / 大众科普）
3. 目标读者（从业者 / 科技爱好者 / 普通大众）
4. 内容侧重（哲学 / 技术 / 案例 / 平衡）
5. 反驳观点（是否需要明确反驳某些论调）
6. 人称视角（第一人称 / 第三人称 / 署名）
7. 输出形式（全文字数 + 翻页版页数）

**阶段二：大纲确认**
- 写文章大纲给用户确认（标题 + 章节结构 + 字数分配）
- 用户确认后才能开始写作

**阶段三：内容创作**
- 三版本输出：
  - 全文版（3000-5000 字）
  - 翻页版（最多 18 页，每页一屏）
  - 截图版（9 页，每页两屏，滚动截图）

**阶段四：HTML 生成与上传**
- 三个 HTML 文件
- 上传飞书云空间
- 发送文件链接给用户

### 版本规范（2026-03-15 更新）⭐

| 版本 | 页数 | 每页高度 | 用途 | 特点 |
|------|------|---------|------|------|
| 全文版 | 单页 | - | 深度阅读 | 桌面友好，可打印 |
| 翻页版 | ≤18 页 | 667px（一屏） | 手机分享 | 点击翻页，触摸滑动 |
| 截图版 | 9 页 | 1334px（两屏） | 滚动截图 | 无导航，方便拼图 |

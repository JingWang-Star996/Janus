## 💡 经验教训

### 浏览器配置遗忘问题
- **问题**：多次忘记使用 OpenClaw 专属浏览器配置
- **原因**：配置仅存储在 MEMORY.md，没有主动加载机制
- **解决**：创建 Skill 文档 + 更新 MEMORY.md + 建立检查清单
- **优先级**：最高（用户多次提醒）

### 飞书图片下载限制
- 飞书图片需要认证权限，image_key 不能直接转换为可访问 URL
- 必须等用户把图片作为消息附件发送才能分析

### 浏览器窗口同步问题
- WSL2 环境中的浏览器与 Windows 侧不是同一窗口
- 用户无法直接看到 AI 操作的浏览器窗口
- 解决方案：截图分享或使用 Windows 本机 Chrome

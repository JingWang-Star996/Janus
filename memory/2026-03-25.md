## 关键技术点

### 飞书消息折叠机制
- 触发条件：视觉行数超过 6-8 行、大量空行、多段列表/表格、300+ 字
- 解决方案：控制在 5 行内、用**加粗**代替换行、减少空行、合并段落

### Browser 工具能力
- ✅ 能用 `browser open + snapshot` 查看公开网页
- ❌ 无法点击交互（登录、按钮等）
- ❌ Cron systemEvent 无法自动调用 Browser

### Cron 任务配置
```json
{
  "name": "金山文档监控",
  "schedule": {"kind": "every", "everyMs": 30000},
  "payload": {"kind": "agentTurn", "message": "..."},
  "sessionTarget": "isolated",
  "delivery": {"mode": "announce"}
}
```

## ❌ 失败的测试

### 条件分支测试

**问题**: 条件判断无法访问 context 变量

**原因**: `eval()` 在局部作用域执行，无法访问传入的 context 参数

**原始代码**:
```javascript
return eval(condition.expression);  // ❌ 无法访问 context
```

**修复方案**:
```javascript
const fn = new Function('context', `return ${condition.expression};`);
return fn(context);  // ✅ 正确传递 context
```

**状态**: 🟢 已修复（见 workflowEngine.js 第 87-98 行）

## 使用规范

### 触发方式
- 直接@对应岗位 Agent 名称
- 描述 HR 问题，系统自动路由

### 协作流程
```
简单问题 → 对应专员 → 解决
复杂问题 → 总监级 → 跨模块协调
战略问题 → CHRO → 高层决策
```

### 输出标准
- 管理层：战略视角 + 执行路径 + 量化指标 + 风险预案
- 总监级：方案完整 + 数据支撑 + 可落地
- 专员级：精准执行 + 专业细节 + 时效保证

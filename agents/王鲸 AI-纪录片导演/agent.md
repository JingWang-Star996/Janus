## 🔧 技术实现

### 日志接入
```javascript
// 接入 Orchestra 日志流
orchestra.on('log', (log) => {
  documentaryRecorder.record(log);
});

// 接入 Agent 日志流
agents.forEach(agent => {
  agent.on('log', (log) => {
    documentaryRecorder.record(log);
  });
});
```

### 纪录片生成
```javascript
// 实时生成纪录片章节
function generateChapter(phase, logs) {
  return `
## ${phase.name}
**时间**：${phase.timestamp}

${narrativeStyle.write(logs)}
`;
}
```

## 🎯 典型案例

**需求**："实现一个背包系统"

**策划需求**：
- 背包有 100 个格子
- 道具可以堆叠（最多 999 个）
- 道具可以排序（按类型、品质、数量）
- 道具可以出售、使用、丢弃

**你的实现思路**：

1. **数据结构**：
```csharp
class Item {
    int id;           // 道具 ID
    int count;        // 数量
    int quality;      // 品质
    ItemType type;    // 类型
}

class Bag {
    Item[] items = new Item[100];  // 100 个格子

    void AddItem(int id, int count) { ... }
    void RemoveItem(int id, int count) { ... }
    void SortByType() { ... }
    void SortByQuality() { ... }
}
```

2. **功能实现**：
   - AddItem：找空格子或堆叠
   - RemoveItem：减少数量，数量为 0 时删除
   - SortByType：按类型排序
   - SortByQuality：按品质排序

3. **UI 实现**：
   - 用 UGUI 实现背包界面
   - 每个格子是一个 ItemSlot
   - 点击格子显示道具详情

4. **优化**：
   - 用对象池管理 ItemSlot（避免频繁创建销毁）
   - 用脏标记（Dirty Flag）减少刷新

5. **测试**：
   - 测试添加道具（空格子、堆叠）
   - 测试删除道具（部分删除、全部删除）
   - 测试排序（按类型、按品质）
   - 测试边界（背包满、道具数量 999）

**输出**：
- 技术方案文档
- 代码（Bag.cs、ItemSlot.cs）
- 自测报告

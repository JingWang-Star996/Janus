## 🎯 典型案例

**需求**："实现一个排行榜系统"

**策划需求**：
- 排行榜按战力排序
- 显示前 100 名
- 每小时更新一次
- 玩家可以查看自己的排名

**你的实现思路**：

1. **数据结构**：
   - 用 Redis 的 Sorted Set 存储排行榜
   - key: "rank:power"
   - score: 战力
   - member: 玩家 ID

2. **协议设计**：
```protobuf
message GetRankRequest {
    int32 start = 1;  // 起始排名
    int32 count = 2;  // 数量
}

message GetRankResponse {
    repeated RankItem items = 1;
}

message RankItem {
    int32 rank = 1;      // 排名
    int64 player_id = 2; // 玩家 ID
    int32 power = 3;     // 战力
    string name = 4;     // 名字
}
```

## 📋 核心能力

### 1. 入职前准备

```
1. 团队角色清单
   - 职位列表（制作人/主程序/主美/策划/后端/前端/美术/测试）
   - 职责说明
   - 联系方式（飞书 open_id）

2. 群聊规则配置
   - @过滤策略（requireMention=true）
   - 人员白名单（allowFrom）
   - 响应规则（只响应@）

3. 权限配置
   - 飞书应用权限
   - 文档/表格访问权限
   - 任务管理权限
```

### 2. 第一天工作流

```
09:00 - 欢迎仪式（可选）
        - 群内介绍
        - 角色定位说明

10:00 - 文档学习清单
        - 世界观文档
        - 系统策划案
        - 技术规范

14:00 - 第一个任务（建立信任）
        - 创建 BUG 收集表
        - 录入已知问题
        - 设置权限

18:30 - 日报提交
        - 工作概览
        - 完成情况
        - 待办事项
```

### 3. 第一周工作流

```
每日：
- 18:30 自动日报
- BUG 跟踪更新
- 群内问题响应

本周目标：
- 接管重复性工作（日报/BUG 跟踪）
- 学习所有核心文档
- 掌握团队术语
- 准确分派任务
```

### 4. 第一个月工作流

```
定时汇报：
- 每日 18:30 日报
- 每周五周报汇总
- 每月进度总结

主动跟进：
- BUG 状态跟踪
- 任务进度询问
- 风险预警

知识沉淀：
- 新人问答索引
- 术语词典
- 流程 SOP
```

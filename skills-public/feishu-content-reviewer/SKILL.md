## 📋 核心能力

### 1. 隐私脱敏审查

自动检查以下内容：

| 类型 | 示例 | 处理方式 |
|------|------|----------|
| **团队成员人名** | 齐翔、陈华强、赖力等 | 替换为职位代号（后端主程、前端主程） |
| **作者本人** | jing、王鲸（文章作者） | ✅ 保留，无需脱敏 |
| 公司名称 | 联合进化公司、XX 科技 | 模糊为"某上市公司"或"合作企业" |
| 项目代号 | ZVP、Yummigo | 模糊为"在研项目"或"某游戏项目" |
| 具体 BUG | 1568 错误、2058939367 | 泛化为"系统错误"或"数据同步问题" |
| 服务器地址 | 外网测试服、正式服 | 模糊为"线上环境"或"测试环境" |
| 内部术语 | DCE 部门、UEC | 替换为通用说法或模糊处理 |
| 具体数据 | 19 万字、12 人团队 | 评估后决定是否保留 |
| **玩法系统名** | 周常副本、日常任务 | 泛化为"玩法模块"、"游戏功能" |
| **具体系统名称** | 竞技场、枪械、宠物 | 泛化为"核心系统"、"功能模块" |
| **策划文档名** | 具体文档标题 | 泛化为"系统策划文档" |
| **具体数值** | 1568 次上限 | 泛化为"次数上限"或评估后保留 |
| **系统机制** | 自动分解、踢登录页 | 泛化为"系统处理"、"重新登录" |

### 2. 风险分级

```
高风险（必须修改）：
- 真实人名（除作者外）
- 具体项目/系统名称
- 具体 BUG 代码/错误日志
- 商业机密信息

中风险（建议修改）：
- 团队规模具体数字
- 具体数值/数据
- 内部术语

低风险（可选优化）：
- 时间信息
- 通用平台名称（飞书、OpenClaw）
- 工作流程描述
```

### 3. 审查流程

```
1. 接收待审核文章
   ↓
2. 逐项检查隐私内容
   ↓
3. 标记问题并分类（高/中/低风险）
   ↓
4. 生成审查报告（表格形式）
   ↓
5. 提供修改建议
   ↓
6. 根据建议修改文章
   ↓
7. 创建飞书文档
   ↓
8. 输出最终版本 + 文档链接
```

## 🔧 使用方法

### 方式一：直接审查

```
用户：帮我审查这篇文章，准备发公众号
[粘贴文章内容]

AI: 
1. 生成审查报告
2. 提供修改建议
3. 询问是否直接修改
4. 修改后创建飞书文档
5. 发送文档链接
```

### 方式二：审查 + 修改

```
用户：这篇文章脱敏处理一下，要发公众号
[粘贴文章内容]

AI:
1. 自动审查
2. 直接修改
3. 创建飞书文档
4. 发送文档链接 + 审查报告
```

### 方式三：仅审查不修改

```
用户：检查一下这篇文章有没有隐私问题
[粘贴文章内容]

AI:
1. 生成审查报告
2. 列出问题清单
3. 提供修改建议
4. 不自动修改（等用户确认）
```

## 📝 输出物

### 1. 审查报告（Markdown 表格）

```markdown
## 内审部审查报告

**审核时间**：2026-03-20 20:30
**审核对象**：《文章标题》
**审核结果**：⚠️ 需修改

### 发现的问题

| 序号 | 原文内容 | 风险等级 | 修改建议 |
|------|----------|----------|----------|
| 1 | 王鲸（制作人） | 高 | 替换为"游戏制作人王鲸"（作者保留） |
| 2 | 齐翔（后端） | 高 | 替换为"后端主程序" |
| 3 | 周常副本 | 高 | 泛化为"玩法模块" |
| ... | ... | ... | ... |

### 修改指导

**必须修改**（高风险）：
- 列出所有必须修改的内容

**建议修改**（中风险）：
- 列出建议修改的内容

**可选修改**（低风险）：
- 列出可选优化的内容

### 内审部意见

**总体评价**：文章整体脱敏较好，但仍需对部分项目专有名词进行泛化处理。

**审查员**：内审部 AI 审查员
```

### 2. 修改后文章（飞书文档）

- 自动创建飞书云文档
- 应用所有修改建议
- 设置合适的标题和格式
- 生成可分享链接

| 第三段 | 齐翔 | 后端主程序 | 保护团队成员隐私 |
| ... | ... | ... | ... |

## ⚙️ 配置项

### 可选配置（在 agent.md 或 skill 配置中）

```yaml
# 作者信息（保留不脱敏）
author_names:
  - jing
  - 王鲸

# 团队成员（需要脱敏）
team_members:
  - 齐翔：后端主程序

high_risk_keywords:
  - 周常副本
  - 联结实验室
  - 幸福进化计划

# 中风险关键词（建议脱敏）
medium_risk_keywords:
  - 19 万字
  - 12 人团队

# 飞书文档配置
feishu_doc:
  folder_token: ""  # 默认创建到我的空间
  default_title: "待审查文章"
```

## 🚀 快速开始

### 方式 1：直接下载技能包

```bash
# 进入技能包目录
cd ~/.openclaw/workspace/skills-public/

# 安装技能包（以 AI 分身定位法为例）
openclaw skills install ai-persona-positioning.skill
```

### 方式 2：克隆整个系列

```bash
git clone https://github.com/starmark996/openclaw-wangjing-skills.git
```

### 方式 3：手动配置

1. 阅读 `SKILL.md` 了解方法论
2. 复制 JSON 配置模板
3. 应用到你的 AI 配置中

## 🎯 使用场景

### 企业 AI 部署

- 权限分级 → 防止信息泄露
- 消息审查 → 合规性保障
- 保密框架 → 保护员工隐私

### 团队管理

- AI 分身定位 → 代表管理者立场
- 指令优化 → 提升执行准确率
- 日报自动化 → 节省汇总时间

### 个人效率

- 临时 Agent → 多任务并行处理
- 指令模板 → 减少沟通成本
- 知识沉淀 → 经验复用

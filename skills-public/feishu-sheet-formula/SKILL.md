# feishu-sheet-formula - 飞书表格公式与配表技能

## 技能描述

专业的飞书电子表格公式与游戏配表技能。提供公式编写、数据引用、配置表管理、配表验证等能力。

**适用场景**：
1. 飞书表格公式编写与调试
2. 游戏配置表设计与维护
3. 跨表数据引用与同步
4. 配表数据验证与检查
5. 配置表结构优化

## 核心能力

### 1. 公式编写

#### 基础公式
- `VLOOKUP()` - 垂直查找
- `HLOOKUP()` - 水平查找
- `INDEX()` + `MATCH()` - 灵活查找组合
- `IF()` / `IFS()` - 条件判断
- `TEXTJOIN()` - 文本合并
- `SPLIT()` - 文本分割
- `ARRAYFORMULA()` - 数组公式
- `IMPORTRANGE()` - 跨表引用

#### 高级公式
- `MAP()` + `LAMBDA()` - 批量处理
- `FILTER()` - 数据筛选
- `QUERY()` - 数据查询
- `REGEXMATCH()` / `REGEXEXTRACT()` - 正则匹配

### 2. 配表规范

#### 配置表结构
| 字段类型 | 说明 | 示例 |
|---------|------|------|
| ID | 唯一标识，数字或字符串 | 1001, "pet_001" |
| Name | 名称，用于显示 | "火焰剑" |
| Type | 类型枚举 | 1=武器，2=防具 |
| Value | 数值 | 100, 1.5 |
| Desc | 描述文本 | "增加攻击力" |
| Ref | 引用其他表 | "monster_001" |

#### 命名规范
- 表名：`config_用途_版本` 如 `config_pet_v1`
- 字段名：驼峰或下划线，统一风格
- ID 格式：`类型_编号` 如 `pet_001`, `item_1001`

### 3. 跨表引用

#### IMPORTRANGE 用法
```
=IMPORTRANGE("表格 URL 或 token", "工作表名!范围")
```

#### 注意事项
1. 首次使用需要授权
2. 范围格式：`"Sheet1!A1:D100"`
3. 大量引用会影响性能

### 4. 数据验证

#### 验证规则
- 必填字段检查
- 数据类型验证
- 引用完整性检查
- 数值范围验证
- 唯一性检查

#### 验证公式示例
```
=IF(ISBLANK(A2), "⚠️ ID 不能为空", "✅")
=IF(COUNTIF(A:A, A2)>1, "⚠️ ID 重复", "✅")
=IF(ISERROR(VLOOKUP(D2, 配置表!A:A, 1, FALSE)), "⚠️ 引用不存在", "✅")
```

## 使用示例

### 示例 1：跨表引用
```
=IFERROR(VLOOKUP(VALUE(INDEX(SPLIT(B5,"*"),2)), IMPORTRANGE("URL","A:D"), 2, FALSE), "")
```

### 示例 2：批量处理
```
=TEXTJOIN("|", TRUE, MAP(SPLIT(B5,"|"), LAMBDA(x, VLOOKUP(VALUE(x), 配置表!A:D, 2, FALSE))))
```

### 示例 3：数据验证
```
=IF(AND(NOT(ISBLANK(A2)), NOT(ISBLANK(B2)), C2>0), "✅", "⚠️ 检查字段")
```

## 常见问题

### Q1: IMPORTRANGE 显示#REF!
A: 首次使用需要授权，点击单元格按提示操作

### Q2: 公式计算慢
A: 减少跨表引用数量，使用缓存列

### Q3: VLOOKUP 找不到数据
A: 检查数据类型是否一致，文本用引号，数字用 VALUE() 转换

### Q4: 配置表冲突
A: 使用唯一 ID，添加重复检查公式

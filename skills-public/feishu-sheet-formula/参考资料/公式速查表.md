# 飞书表格公式速查表

## 基础公式

### 查找引用
| 公式 | 说明 | 示例 |
|------|------|------|
| `VLOOKUP(值，范围，列，精确)` | 垂直查找 | `=VLOOKUP(A2, 配置表!A:D, 2, FALSE)` |
| `HLOOKUP(值，范围，行，精确)` | 水平查找 | `=HLOOKUP(A2, 配置表!1:4, 2, FALSE)` |
| `INDEX(范围，行，列)` | 返回指定位置值 | `=INDEX(A1:D100, 5, 2)` |
| `MATCH(值，范围，类型)` | 返回位置 | `=MATCH(A2, B:B, 0)` |
| `IMPORTRANGE("URL","范围")` | 跨表引用 | `=IMPORTRANGE("https://...","Sheet1!A1:D100")` |

### 文本处理
| 公式 | 说明 | 示例 |
|------|------|------|
| `TEXTJOIN("分隔符",TRUE,范围)` | 合并文本 | `=TEXTJOIN("|",TRUE,A1:A10)` |
| `SPLIT(文本，"分隔符")` | 分割文本 | `=SPLIT(A1,"|")` |
| `LEFT(文本，长度)` | 左侧字符 | `=LEFT(A1,3)` |
| `RIGHT(文本，长度)` | 右侧字符 | `=RIGHT(A1,3)` |
| `MID(文本，开始，长度)` | 中间字符 | `=MID(A1,2,5)` |
| `LEN(文本)` | 文本长度 | `=LEN(A1)` |
| `TRIM(文本)` | 去除空格 | `=TRIM(A1)` |
| `SUBSTITUTE(文本，旧，新)` | 替换文本 | `=SUBSTITUTE(A1,"|",",")` |

### 逻辑判断
| 公式 | 说明 | 示例 |
|------|------|------|
| `IF(条件，真，假)` | 条件判断 | `=IF(A2>100,"高","低")` |
| `IFS(条件 1，值 1，条件 2，值 2,...)` | 多条件判断 | `=IFS(A2>90,"A",A2>80,"B",TRUE,"C")` |
| `AND(条件 1，条件 2)` | 与逻辑 | `=IF(AND(A2>0,A2<100),"OK","NG")` |
| `OR(条件 1，条件 2)` | 或逻辑 | `=IF(OR(A2="Y",A2="是"),"通过","不通过")` |
| `IFERROR(公式，错误值)` | 错误处理 | `=IFERROR(VLOOKUP(...),"未找到")` |
| `ISBLANK(单元格)` | 是否空白 | `=IF(ISBLANK(A2),"⚠️ 必填","✅")` |

### 数学计算
| 公式 | 说明 | 示例 |
|------|------|------|
| `SUM(范围)` | 求和 | `=SUM(A1:A100)` |
| `AVERAGE(范围)` | 平均值 | `=AVERAGE(A1:A100)` |
| `MAX(范围)` | 最大值 | `=MAX(A1:A100)` |
| `MIN(范围)` | 最小值 | `=MIN(A1:A100)` |
| `COUNT(范围)` | 计数（数字） | `=COUNT(A1:A100)` |
| `COUNTA(范围)` | 计数（非空） | `=COUNTA(A1:A100)` |
| `COUNTIF(范围，条件)` | 条件计数 | `=COUNTIF(A1:A100,">100")` |
| `SUMIF(范围，条件，求和范围)` | 条件求和 | `=SUMIF(A1:A100,">100",B1:B100)` |

### 数组函数（新）
| 公式 | 说明 | 示例 |
|------|------|------|
| `MAP(数组，LAMBDA(x,公式))` | 批量处理 | `=MAP(A1:A10,LAMBDA(x,VLOOKUP(x,配置表!A:B,2,0)))` |
| `FILTER(范围，条件)` | 筛选数据 | `=FILTER(A1:D100,C1:C100="激活")` |
| `SORT(范围，列，升序)` | 排序 | `=SORT(A1:D100,2,TRUE)` |
| `UNIQUE(范围)` | 去重 | `=UNIQUE(A1:A100)` |
| `SEQUENCE(行数，列数)` | 生成序列 | `=SEQUENCE(10,1)` |

## 配表专用公式

### 数据验证
```
// 检查 ID 是否为空
=IF(ISBLANK(A2), "⚠️ ID 不能为空", "✅")

// 检查 ID 是否重复
=IF(COUNTIF(A:A, A2)>1, "⚠️ ID 重复", "✅")

// 检查必填字段
=IF(OR(ISBLANK(A2),ISBLANK(B2),ISBLANK(C2)), "⚠️ 必填字段缺失", "✅")

// 检查数值范围
=IF(OR(D2<0,D2>100), "⚠️ 数值超出范围", "✅")

// 检查引用是否存在
=IF(ISERROR(VLOOKUP(E2, 配置表!A:A, 1, FALSE)), "⚠️ 引用不存在", "✅")

// 综合验证
=IF(AND(NOT(ISBLANK(A2)),NOT(ISBLANK(B2)),COUNTIF(A:A,A2)=1,D2>=0), "✅", "⚠️ 请检查")
```

### 跨表引用
```
// 基础跨表引用
=IMPORTRANGE("https://feishu.cn/sheets/TOKEN", "Sheet1!A1:D100")

// 跨表 VLOOKUP
=VLOOKUP(A2, IMPORTRANGE("URL","配置表!A:D"), 2, FALSE)

// 带错误处理的跨表引用
=IFERROR(IMPORTRANGE("URL","Sheet1!A1"),"加载失败")
```

### 复杂数据处理
```
// 分割后批量查找并合并
=TEXTJOIN("|", TRUE, MAP(SPLIT(B5,"|"), LAMBDA(x, 
  IFERROR(VLOOKUP(VALUE(x), 配置表!A:D, 2, FALSE), "")
)))

// 提取 ID 并查找（处理 "类型*ID" 格式）
=VLOOKUP(VALUE(INDEX(SPLIT(B5,"*"),2)), IMPORTRANGE("URL","A:D"), 2, FALSE)

// 多条件查找
=INDEX(配置表!C:C, MATCH(1, (配置表!A:A=A2)*(配置表!B:B=B2), 0))

// 动态列查找
=INDEX(配置表!A:Z, MATCH(A2, 配置表!A:A, 0), MATCH("目标列名", 配置表!1:1, 0))
```

### 统计汇总
```
// 按类型统计数量
=COUNTIF(配置表!B:B, "宠物")

// 按类型汇总数值
=SUMIF(配置表!B:B, "宠物", 配置表!D:D)

// 多条件统计
=COUNTIFS(配置表!B:B, "宠物", 配置表!C:C, "SR")

// 多条件汇总
=SUMIFS(配置表!D:D, 配置表!B:B, "宠物", 配置表!C:C, "SR")

// 去重计数
=COUNTA(UNIQUE(配置表!A:A))
```

## 性能优化技巧

1. **减少 IMPORTRANGE 数量**
   - 合并多个引用到一个辅助表
   - 使用缓存列存储引用结果

2. **避免整列引用**
   - ❌ `VLOOKUP(A2, 配置表!A:D, 2, 0)`
   - ✅ `VLOOKUP(A2, 配置表!A2:D1000, 2, 0)`

3. **使用辅助列**
   - 复杂计算拆分成多列
   - 每列做一步处理

4. **禁用自动计算**
   - 大量数据导入时临时禁用
   - 完成后再启用

## 调试技巧

1. **分步验证**
   ```
   // 第一步：检查数据是否存在
   =COUNTIF(配置表!A:A, A2)
   
   // 第二步：检查数据类型
   =ISNUMBER(A2)
   
   // 第三步：执行查找
   =VLOOKUP(A2, 配置表!A:D, 2, 0)
   ```

2. **使用辅助列显示中间结果**
   ```
   // 显示分割结果
   =SPLIT(B5,"|")
   
   // 显示查找结果
   =VLOOKUP(...)
   ```

3. **错误定位**
   ```
   =IFERROR(公式，"错误："&ERROR_TYPE())
   ```

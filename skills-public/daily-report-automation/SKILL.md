## 📖 技能描述

自动抓取群聊消息，提取进度信息，生成日报/周报，节省人工汇总时间，数据客观准确。

---

## 🎯 核心主张

> **"让 AI 写日报，让人做决策"**

**问题**：人工汇总日报耗时、主观、易遗漏  
**解决**：自动抓取 + 智能提取 + 模板生成

---

## 🏗️ 自动化流程

```
定时触发 (每日 18:30)
    ↓
抓取群聊消息 (多个群)
    ↓
提取关键信息
    ├─ 人员 (@提及)
    ├─ 事项 (任务/进度)
    ├─ 时间 (截止时间)
    └─ 状态 (完成/进行中/待办)
    ↓
分类汇总
    ├─ 按项目
    ├─ 按人员
    └─ 按状态
    ↓
生成日报 (Markdown/HTML/飞书文档)
    ↓
发送汇报 (私聊/群聊/飞书)
    ↓
记录日志
```

## 📋 信息提取规则

### 人员识别

```json
{
  "extraction_rules": {
    "person": {
      "pattern": "@([\\u4e00-\\u9fa5]+)",
      "source": "mention_list",
      "fallback": "sender_name"
    }
  }
}
```

### 进度识别

```json
{
  "progress_keywords": {
    "completed": ["完成", "搞定", "ok", "✅"],
    "in_progress": ["进行中", "搞着", "ing", "🟡"],
    "pending": ["待办", "计划", "todo", "⏳"],
    "blocked": ["卡住", "问题", "help", "🔴"]
  }
}
```

### 时间识别

```json
{
  "time_keywords": {
    "today": ["今天", "今日", "下班前"],
    "tomorrow": ["明天", "明日", "明天下班前"],
    "deadline": ["截止", "ddl", "之前"]
  }
}
```

## ⚠️ 注意事项

1. **隐私保护**：不记录敏感信息（请假理由、薪资等）
2. **准确性**：关键数据需人工确认
3. **时效性**：按时发送，避免延迟
4. **格式**：根据接收方调整（飞书/微信/邮件）

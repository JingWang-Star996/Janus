## 🔧 核心技巧详解

### 技巧 01：场景 + 规则 + 示例

**❌ 模糊指令**：
> "处理好群聊消息"

**✅ 优化指令**：
```
场景：群聊回复
规则：
1. 被@时正常回复
2. 没@你保持沉默
3. 只发执行结果，不发思考过程
示例：
- @你"进度怎么样" → "已完成 80%，明天交付"
- 群内普通聊天 → 不回复
```

**效果**：准确率 +80%

---

### 技巧 02：边界条件明确

**❌ 模糊边界**：
> "注意保密"

**✅ 优化指令**：
```
保密边界：
- ✅ 可对外：工作安排、文档位置、会议时间
- ❌ 不对外：请假理由、薪资数据、优化名单
- ⚠️ 需确认：离职原因、岗位调整

越权处理：
- 下属问隐私 → "抱歉，这是个人隐私"
- 平级问决策 → "需要XXX确认"
- 记录审查日志
```

**效果**：合规性 +60%

---

### 技巧 03：反馈循环

**指令模板**：
```
任务：XXX
执行 → 检查结果 → 修正 → 固化到配置

示例：
1. AI 回复群聊 → 用户指出问题 → 修正规则 → 写入 MEMORY.md
2. 每周回顾：哪些指令有效，哪些需要优化
```

**效果**：迭代速度 +50%

---

### 技巧 04：知识沉淀

**沉淀位置**：
- MEMORY.md - 长期记忆
- 技能配置 - 可复用规则
- 话术库 - 标准回复

**示例**：
```markdown
## 群聊回复规则（2026-03-24）

来源：XXX明确要求

内容：
> 群聊没有@你的不要自主响应回复

执行：
- ✅ 被@时 → 正常回复
- ❌ 群内普通聊天 → 不回复
```

**效果**：复用率 +70%

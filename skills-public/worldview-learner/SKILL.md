# worldview-learner - 世界观/文档快速学习 Skill

**版本**：1.0.0  
**创建日期**：2026-03-20  
**作者**：jing（OpenClaw 实践者）  
**适用场景**：游戏/内容行业的文档快速学习与知识沉淀

---

## 🎯 触发词

```
- "学习这个文档"
- "读一下策划案"
- "总结这个系统"
- "学习世界观"
- "文档学习"
- "知识沉淀"
- "术语提取"
```

---

## 📋 核心能力

### 1. 文档读取
- 飞书文档/表格/知识库
- 本地文件（docx/xlsx/pdf）
- 链接内容抓取

### 2. 结构化总结
- 系统名称
- 核心设计
- 关键机制
- 数据表结构
- UI/交互需求

### 3. 术语提取
- 专有名词列表
- 术语解释
- 使用场景

### 4. 记忆存储
- 长期记忆更新
- 术语词典
- 问答索引

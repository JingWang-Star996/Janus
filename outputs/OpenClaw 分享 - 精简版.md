# OpenClaw 实战分享

**需求**：做个 AI 使用情况收集表，让团队成员填写后生成统一文本。

**成果**：2 小时交付智能表单网页（单文件，移动端完美适配）

**核心功能**：
- 选岗位→动态显示场景
- 勾选使用场景和工具
- 一键生成统一格式文本
- 一键复制发送

**OpenClaw 使用心得**：
1. 直接说需求，不用管技术实现
2. 及时反馈，快速迭代（6 次）
3. BUG 说现象，AI 快速定位（5 分钟修复）
4. 文件直接要，自动发送
5. 文档一句话，自动生成

**数据**：2 小时 / 6 次迭代 / 900 行代码 / 1 个 BUG

**适合场景**：快速原型、工具脚本、数据分析、自动化任务

🚀 5 步快速部署

  • 安装 Node.js 访问 nodejs.org 下载安装包，一路下一步即可
    node -v # 验证安装，显示版本号即成功
  • 安装 OpenClaw 打开终端/命令行，执行安装命令
    npm install -g openclaw
  • 配置 AI 模型 使用阿里云百炼或其他 AI 服务
    openclaw configure --section models
  • 配置消息渠道 连接 Telegram 或其他消息平台
    openclaw configure --section channels
  • 启动服务 启动 Gateway，开始使用
    openclaw gateway start

🎉 完成！现在你可以开始和 AI 助手对话了！

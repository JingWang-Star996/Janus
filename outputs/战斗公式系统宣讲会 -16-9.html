🏃 英雄属性（F001-F008）

F001 英雄最终攻击力：
(等级攻击加成 + 技能固定攻击加成 + 基础攻击力 + 训练攻击加成) × (1 + 攻击力百分比加成) >> 0
F003 综合战力：
Math.ceil(4 × (最终攻击力 × 暴击期望 / 冷却倍率 + 最大生命值 × (1 + 减伤率)))

设计要点

  • >>0 位运算向下取整
  • 百分比加成先累加再统一乘算
  • 战力公式：攻击力权重4，生命值权重1
  • 冷却倍率最小值保护0.1

💰 掉落获取（F068-F070）

类型 公式 说明
经验 基础经验 × 经验倍率 × 难度倍率 向下取整
金币 基础金币 × 金币倍率 × 难度倍率 向下取整
掉落物 基础数量 × 掉落倍率 向下取整

难度倍率设计

  • 由挑战模式配置决定
  • 普通模式倍率为1
  • 高难度模式倍率>1，奖励更丰厚

👹 敌人属性（F071-F074）

公式 名称 公式
F071 最终攻击力 基础攻击力 × 难度攻击倍率
F072 最终生命值 基础生命值 × 难度生命倍率
F073 移动速度 基础速度 × 难度速度倍率 × 30
F074 攻击间隔 基础间隔 / 难度攻速倍率

设计意图

  • 通过难度倍率调整游戏挑战性
  • 高难度下敌人更强，奖励倍率也更高
  • 保持游戏平衡性

⚔️ 伤害计算（F075-F080）

if(暴击类型 = SECKILL) {
  伤害 = 目标当前生命值
} else if(暴击类型 = SUPER_CRIT) {
  伤害 = 基础伤害 × 超级暴击倍数
} else if(暴击类型 = CRIT) {
  伤害 = 基础伤害 × 暴击倍数
} else {
  伤害 = 基础伤害
}

等级加成

  • 📚 技能等级加成（F079）：技能当前等级 - 1
  • 🔫 武器等级加成（F080）：武器当前等级 - 1
  • 1 级加成为0，每级提升+1

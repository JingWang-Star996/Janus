🏃 英雄属性公式（F001-F008）

核心公式示例

F001 英雄最终攻击力：
(等级攻击加成 + 技能固定攻击加成 + 基础攻击力 + 训练攻击加成) × (1 + 攻击力百分比加成) >> 0
F003 综合战力：
Math.ceil(4 × (最终攻击力 × 暴击期望 / 冷却倍率 + 最大生命值 × (1 + 减伤率)))

设计要点

  • >>0 位运算向下取整，保证数值为整数
  • 百分比加成先累加再统一乘算，避免多次乘法误差
  • 战力公式中攻击力权重为4，生命值权重为1
  • 冷却倍率最小值保护为0.1，防止除零错误

💚 回血机制（F009-F015）

回血类型

🍖 吃肉回血

掉落值/100 + 百分比 × 最大生命值

✨ 技能回血

百分比/100 × 最大生命值 × 总加成

⬆️ 升级回血

百分比 × 最大生命值

💊 复活回血

百分比/100 × 最大生命值 +3 秒无敌

特殊机制

  • 过量回血转护盾（F013）：溢出量转换为能量护盾
  • 护盾优先级（F014）：普通护盾 > 能量护盾 > 生命值
  • 濒死无敌（F015）：血量归零时触发，仅一次

🎯 暴击判定（F016-F018）

判定优先级

秒杀 > 超级暴击 > 普通暴击 > 普通伤害

伤害倍率

  • 秒杀：伤害 = 目标当前生命值（直接击杀）
  • 超级暴击：伤害 = 基础 × 3.0
  • 普通暴击：伤害 = 基础 × 2.0

判定流程

秒杀：if(random(0,100) <= 秒杀概率) → SECKILL
超暴：if(random(0,100) < 超级暴击概率) → SUPER_CRIT
普暴：if(random(0,100) < 暴击概率) → CRIT

🔫 子弹系统（F019-F031）

核心属性

💥 攻击力

F019：子弹伤害标记>0 时使用 F004 公式

📏 形状尺寸

F021-F024：矩形/圆形/扇形，×30 转像素

⚡ 速度

F026：基础速度 × 倍率 × 30（像素/秒）

🎯 射程

F027：基础射程 × 倍率 × 30（像素）

特殊效果

  • 穿透（F028）：基础穿透 + 加成，影响可击中敌人数量
  • 弹射（F029）：基础弹射 + 加成，连续击中敌人
  • 分裂（F030）：基础分裂 + 加成，消失时生成新子弹

⚔️ 武器技能（F032-F049）

武器核心

  • 冷却倍率（F032）：1 - 冷却减免值，最小值 0.1
  • 发射间隔（F038）：基础间隔 / 攻速倍率
  • 子弹数量（F039）：基础数量 + 加成
  • 触发概率（F040）：Math.min(100, 基础 + 加成)

技能核心

  • 冷却时间（F042）：基础冷却 × (1 - 技能冷却减免)
  • 持续时间（F043）：基础持续 × 持续时间倍率
  • 作用范围（F044）：基础范围 × 倍率 × 30

倍率系统

经验倍率（F045）、金币倍率（F046）、掉落倍率（F047）、磁铁范围（F048）

🛡️ 战斗增益（F050-F067）

防御类

  • 无敌时长（F050）：基础时长 + 加成（秒）
  • 护盾层数（F051）：每层抵挡一次伤害
  • 能量护盾值（F052）：抵挡等量伤害数值

进攻类

  • 伤害倍率：范围（F055）、单体（F056）、精英（F057）、Boss（F058）
  • 最终伤害加成（F062）：基础伤害 × (1 + 加成百分比)
  • 技能/武器伤害倍率（F063-F064）：针对性增伤

抗性类

  • 击退抗性（F059）：减少击退距离
  • 控制抗性（F060）：减少控制时长
  • 伤害减免（F061）：百分比减少受到伤害

💰 掉落获取（F068-F070）

获取公式

F068 经验获取量：
基础经验 × 经验倍率 × 难度倍率（向下取整）
F069 金币获取量：
基础金币 × 金币倍率 × 难度倍率（向下取整）
F070 掉落物数量：
基础数量 × 掉落倍率（向下取整）

难度倍率

  • 由挑战模式配置决定
  • 普通模式倍率为1
  • 高难度模式倍率>1，奖励更丰厚

👹 敌人属性（F071-F074）

敌人基础属性

  • 最终攻击力（F071）：基础攻击力 × 难度攻击倍率
  • 最终生命值（F072）：基础生命值 × 难度生命倍率
  • 移动速度（F073）：基础速度 × 难度速度倍率 × 30
  • 攻击间隔（F074）：基础间隔 / 难度攻速倍率

难度倍率

所有敌人属性倍率由挑战模式配置决定，普通模式均为1

设计意图

  • 通过难度倍率调整游戏挑战性
  • 高难度下敌人更强，但奖励倍率也更高
  • 保持游戏平衡性

⚔️ 伤害计算（F075-F080）

伤害计算流程

if(暴击类型 = SECKILL) {
  伤害 = 目标当前生命值 // 秒杀
} else if(暴击类型 = SUPER_CRIT) {
  伤害 = 基础伤害 × 超级暴击倍数 // 超暴
} else if(暴击类型 = CRIT) {
  伤害 = 基础伤害 × 暴击倍数 // 普暴
} else {
  伤害 = 基础伤害 // 普通
}

等级加成

  • 技能等级加成（F079）：技能当前等级 - 1
  • 武器等级加成（F080）：武器当前等级 - 1
  • 1 级加成为0，每级提升+1

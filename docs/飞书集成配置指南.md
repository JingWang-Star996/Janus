## 🚀 配置步骤

### 第 1 步：登录飞书开放平台

**网址**：https://open.feishu.cn/

**操作**：
1. 用飞书 App 扫码登录
2. 进入"企业自建应用"
3. 点击"创建应用"

---

### 第 2 步：创建应用

**应用信息**：
```
应用名称：OpenClaw 助手
应用图标：可选
应用描述：AI 助手机器人
```

**创建后获得**：
- **App ID**：`cli_xxxxxxxxxxxxx`
- **App Secret**：`xxxxxxxxxxxxxxxxx`

**⚠️ 重要**：复制并保存这两个值！

---

### 第 3 步：配置机器人

**在应用管理页面**：

1. 点击"添加能力" → "机器人"
2. 配置机器人名称：`AI 助手`
3. 配置机器人头像（可选）
4. 开启"消息卡片"功能

---

### 第 4 步：配置权限

**需要的权限**：

| 权限 | 用途 | 必须 |
|------|------|------|
| `im:message` | 发送和接收消息 | ✅ |
| `im:chat` | 访问会话信息 | ✅ |
| `contact:user` | 获取用户信息 | ⚠️ 可选 |

**操作**：
1. 点击"权限管理"
2. 搜索上述权限
3. 点击"申请"
4. 提交审核（通常自动通过）

---

### 第 5 步：配置事件订阅

**订阅的事件**：

| 事件 | 用途 |
|------|------|
| `im.message.receive_v1` | 接收消息 |
| `im.chat.update_v1` | 会话更新 |

**操作**：
1. 点击"事件订阅"
2. 点击"添加事件"
3. 选择上述事件
4. 填写接收地址（Webhook URL）

**Webhook URL**（暂时留空，后续配置）：
```
http://你的服务器IP:8000/feishu/webhook
```

---

### 第 6 步：发布应用

**操作**：
1. 点击"版本管理与发布"
2. 点击"发布为正式版本"
3. 填写版本说明
4. 提交审核

**审核时间**：通常 1-3 个工作日

## ⚠️ 常见问题

### Q1: 权限申请失败

**原因**：企业账号限制

**解决**：
- 联系企业管理员开通权限
- 或使用个人开发者账号

---

### Q2: Webhook 收不到消息

**检查**：
1. 服务器是否有公网 IP
2. 防火墙是否开放端口
3. Webhook URL 是否正确

**解决**：
- 使用内网穿透工具（ngrok）
- 配置云服务器

---

### Q3: 消息格式错误

**飞书消息格式**：
```json
{
  "msg_type": "text",
  "content": {
    "text": "消息内容"
  }
}
```

**确保 OpenClaw 能解析此格式**

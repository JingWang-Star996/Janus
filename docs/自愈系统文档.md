# OpenClaw 自愈系统文档

## 🎯 设计目标

**核心问题**：Gateway 挂掉后 AI 失联，无法自救，必须人工介入。

**解决方案**：建立独立于 AI 会话的自愈系统，通过 systemd timer 自动监控和修复。

## 🔧 组件说明

### 1. 健康检查脚本 (`healthcheck.sh`)

**检查项**：
- ✅ systemd 服务状态
- ✅ Gateway RPC 响应
- ✅ Node 进程存在性

**修复级别**：
1. **级别 1**：Gateway 无响应 → `openclaw gateway restart`
2. **级别 2**：systemd 停止 → `systemctl --user restart`
3. **级别 3**：进程丢失 → 清理僵尸进程 + 重启
4. **级别 4**：完全恢复 → 冷启动

**特性**：
- 防并发锁（避免多个检查同时运行）
- 最多重试 3 次
- 重试间隔 60 秒
- 详细日志记录

### 2. systemd Timer

**配置**：
```ini
[Timer]
OnBootSec=5min        # 启动后 5 分钟首次检查
OnUnitActiveSec=30min # 每 30 分钟检查一次
Persistent=true       # 错过执行时间后补执行
```

**优势**：
- 不依赖 AI 会话
- 不依赖 Cron
- 系统重启后自动恢复
- 低资源占用
- **静默运行，不打扰用户**

### 3. Gateway 服务

**配置**：
```ini
[Service]
Restart=always        # 总是自动重启
RestartSec=10         # 重启间隔 10 秒
Environment="PATH=..." # 完整 PATH 环境变量
```

## 🚨 故障处理

### Gateway 挂了怎么办？

**自动修复**：等待最多 5 分钟，systemd timer 会自动检测并修复。

**手动修复**：
```bash
systemctl --user restart openclaw-gateway
```

### 自检脚本挂了怎么办？

**检查 timer 状态**：
```bash
systemctl --user status openclaw-healthcheck.timer
```

**手动触发**：
```bash
systemctl --user start openclaw-healthcheck.service
```

**查看日志**：
```bash
journalctl --user -u openclaw-healthcheck -n 20
```

### 完全失联怎么办？

**WSL2 重启**：
```bash
# Windows PowerShell
wsl --shutdown
# 然后重新打开 WSL
```

**systemd 重启**：
```bash
systemctl --user daemon-reexec
```

## 📝 维护说明

### 修改检查频率
编辑：`~/.config/systemd/user/openclaw-healthcheck.timer`
```ini
OnUnitActiveSec=5min  # 改为需要的间隔
```
然后：
```bash
systemctl --user daemon-reload
systemctl --user restart openclaw-healthcheck.timer
```

### 禁用自愈系统
```bash
systemctl --user stop openclaw-healthcheck.timer
systemctl --user disable openclaw-healthcheck.timer
```

### 启用自愈系统
```bash
systemctl --user enable openclaw-healthcheck.timer
systemctl --user start openclaw-healthcheck.timer
```

## 🆘 紧急联系

如果自愈系统无法修复问题，请手动执行：

```bash
# 1. 查看状态
openclaw gateway status

# 2. 重启 Gateway
openclaw gateway restart

# 3. 如果还不行，完全重启
systemctl --user restart openclaw-gateway

# 4. 查看日志排查
tail -100 /tmp/openclaw/openclaw-$(date +%Y-%m-%d).log
```

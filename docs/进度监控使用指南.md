# 📊 AI 助手进度监控使用指南

**版本**：v1.0  
**更新日期**：2026-03-13

---

## 🎯 核心功能

实时查看 AI 助手的工作状态和任务进度。

---

## 🚀 快速开始

### 1. 启动 HTTP 服务器

```bash
cd /home/z3129119/.openclaw/workspace
python3 -m http.server 8000
```

### 2. 打开网页

浏览器访问：
```
http://localhost:8000/agent-status.html
```

---

## 💡 使用方法

### 默认状态：监控暂停（0 Token 消耗）

网页显示：
```
⏸️ 监控已暂停
💡 提示：对 AI 说"开始任务：XXX"来启用监控
```

---

### 启用监控

**对 AI 说**：
```
开始任务：帮我做 XXX
```

**AI 会**：
1. 写入 `status.json`，设置 `enabled: true`
2. 网页自动显示任务状态
3. 每 5 秒自动刷新

**示例**：
```
你：开始任务：生成双周汇报 PPT

AI 写入 status.json：
{
  "enabled": true,
  "current_task": "🎮 生成双周汇报 PPT",
  "progress": 0,
  "status": "working",
  "emoji": "🎮"
}
```

---

### 任务进行中

网页自动显示：
```
⚡ 工作中
🎮 生成双周汇报 PPT
进度：50%
预计：2026-03-13 16:30

当前任务详情：
📊 读取需求清单 Excel ✓
📋 整理功能模块 ✓
🎨 生成 HTML PPT ⚡
📤 发送给用户 ⏳
```

---

### 任务结束

**AI 会自动**：
1. 更新 `status.json`，设置 `enabled: false`
2. 标记任务为已完成
3. 网页显示"监控已暂停"

**你也可以说**：
```
任务结束
```

---

## 📋 常用命令

| 命令 | 效果 |
|------|------|
| `开始任务：XXX` | 启用监控，开始新任务 |
| `任务结束` | 完成当前任务，关闭监控 |
| `更新状态` | 手动更新进度 |
| `查看进度` | 显示当前任务状态 |

---

## 🎨 网页功能

### 状态徽章

| 状态 | 显示 | 说明 |
|------|------|------|
| ⚡ 工作中 | 🟠 橙色 | AI 正在执行任务 |
| ✅ 已完成 | 🟢 绿色 | 任务完成 |
| ⏸️ 待机中 | ⚪ 灰色 | 无任务 |
| ⏸️ 监控已暂停 | ⚫ 深灰 | 监控关闭，0 Token 消耗 |

### 统计数据

- ✅ 已完成：历史完成任务数
- ⚡ 进行中：当前任务数（0 或 1）
- ⏳ 待处理：排队任务数
- 🎯 成功率：完成任务/总任务

## 💰 Token 消耗优化

### 监控开启时

```
每 5 秒轮询一次
约 50 tokens/次
约 864,000 tokens/天
约 ¥1.73/天
```

### 监控关闭时

```
0 Token 消耗 ✅
```

### 最佳实践

1. **只在需要时开启监控**
   - 做复杂任务时开启
   - 简单对话不开启

2. **任务结束及时关闭**
   - AI 会自动关闭
   - 也可以手动说"任务结束"

3. **批量任务统一监控**
   - 多个小任务合并为一个大任务
   - 减少开关次数

## ⚠️ 注意事项

1. **网页需要手动刷新**（第一版）
   - 按 F5 刷新
   - 或点击右下角🔄按钮
   - 未来版本会支持自动刷新

2. **HTTP 服务器需要保持运行**
   ```bash
   # 后台运行
   nohup python3 -m http.server 8000 &
   ```

3. **status.json 文件不要删除**
   - AI 会自动创建
   - 删除后网页无法显示状态

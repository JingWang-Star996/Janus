# 搜索技能安装指南

## 问题

ClawHub 服务器限流，无法直接安装搜索技能。

## 已尝试方案

### ❌ ClawHub 安装（限流）
```bash
clawhub install baidu-search
clawhub install multi-search-engine
```
错误：`Rate limit exceeded`

## 替代方案

### 方案一：手动百度搜索（临时）

直接访问百度搜索，复制结果给 AI。

### 方案二：配置 Brave Search API

1. 注册：https://api-dashboard.search.brave.com/register
2. 获取 API Key
3. 配置：
   ```bash
   openclaw configure --section web
   ```
4. 输入 API Key

**免费额度**：每月 $5（约 1000 次搜索）

### 方案三：使用阿里云百炼搜索

当前使用 `bailian/qwen3.5-plus` 模型，可能内置搜索能力。

需要确认：
- 百炼是否提供搜索插件
- 如何在配置中启用

### 方案四：等待 ClawHub 限流解除

稍后再试：
```bash
clawhub install baidu-search
```

## 推荐步骤

1. **立即**：用方案一（手动搜索）
2. **短期**：配置 Brave Search API（5 分钟）
3. **长期**：等待 ClawHub 恢复后安装百度技能

## 🚀 快速安装

### 方法 1：自动安装脚本（推荐）

```bash
# 进入工作区
cd /home/z3129119/.openclaw/workspace

# 运行安装脚本
bash hooks/install-status-plugin.sh

# 按提示编辑配置文件
nano ~/.openclaw/openclaw.json

# 重启 Gateway
openclaw gateway restart
```

---

### 方法 2：手动安装

**步骤 1：复制插件文件**

```bash
cp /home/z3129119/.openclaw/workspace/hooks/openclaw-status-plugin.py \
   ~/.nvm/versions/node/v24.14.0/lib/node_modules/openclaw/hooks/

cp /home/z3129119/.openclaw/workspace/hooks/auto-status-hook.py \
   ~/.nvm/versions/node/v24.14.0/lib/node_modules/openclaw/hooks/
```

**步骤 2：修改 OpenClaw 配置**

编辑 `~/.openclaw/openclaw.json`：

```json
{
  "plugins": [
    "status-monitor"
  ],
  "hooks": {
    "on_message_received": ["auto-status-hook.on_message_received"],
    "on_response_sent": ["auto-status-hook.on_response_sent"]
  }
}
```

**步骤 3：重启 Gateway**

```bash
openclaw gateway restart
```

```python
task_keywords = [

    '写一个', '做一个', '开发', '实现', '配置', '设置'
```

**完成任务**：
```python
done_keywords = [
    '完成了', '做好了', '已经', '搞定', '完毕',
    '✅', '完成', '好了', '好了！'
]
```

### 自定义关键词

编辑 `openclaw-status-plugin.py`：

```python
# 添加你的关键词
self.task_keywords = [
    '开始任务', '帮我',  # 现有
    '你的关键词'         # 新增
]
```

## 🔍 故障排查

### Q1: 插件未加载

**检查**：
```bash
# 查看 Gateway 日志
tail -f ~/.openclaw/logs/gateway.log | grep StatusMonitor
```

**解决**：
- 确认插件文件在正确位置
- 确认配置已添加
- 重启 Gateway

---

### Q2: 状态不更新

**检查**：
```bash
# 查看 status.json
cat /home/z3129119/.openclaw/workspace/status.json
```

**解决**：
- 检查文件权限：`chmod 644 status.json`
- 检查目录权限：`chmod 755 workspace/`

---

### Q3: 网页不显示

**检查**：
```bash
# 确认 HTTP 服务器运行
ps aux | grep "http.server"
```

**解决**：
```bash
# 重启 HTTP 服务器
cd /home/z3129119/.openclaw/workspace
python3 -m http.server 8000
```

## 📊 监控逻辑

### 工作流程

```
用户消息
  ↓
检测关键词
  ↓
是任务？→ 写入 status.json (enabled: true)
  ↓
AI 执行任务
  ↓
检测完成关键词
  ↓
是完成？→ 写入 status.json (enabled: false)
  ↓
网页自动刷新显示
```

### 状态转换

```
待机中 (enabled: false)
  ↓ 用户说"帮我 XXX"
工作中 (enabled: true, progress: 0%)
  ↓ AI 执行任务
工作中 (enabled: true, progress: 50%)
  ↓ AI 说"完成了"
已完成 (enabled: false, progress: 100%)
  ↓ 5 秒后
待机中 (enabled: false)
```

## ⭐ 推荐安装

### 🌟 强烈推荐（2 个）

#### 1. daily-summary v1.0.0
- **用途**：自动生成每日学习总结，包含日期、成就、任务状态、计划和 token 使用统计
- **依赖**：阿里云百炼 API（已有）
- **推荐理由**：
  - ✅ 与现有工作流完美集成
  - ✅ 自动追踪 token 使用（符合用户优化原则）
  - ✅ 自动生成日报，节省时间
- **安装命令**：`clawhub install daily-summary`

#### 2. godot-bridge v1.2.0
- **用途**：通过 HTTP/WebSocket 远程控制 Godot 4.x 游戏，支持 2D/3D 图形、UI 特效、动画、粒子、物理等 100+ CLI 命令
- **依赖**：Godot 4.x（可选）
- **推荐理由**：
  - ✅ 游戏开发神器（符合你的游戏项目兴趣）
  - ✅ 100+ CLI 命令，功能强大
  - ✅ 可用于快速原型开发
- **安装命令**：`clawhub install godot-bridge`

### ⚠️ 值得考虑（3 个）

# 🍠 上只当麻的助手 - 头像生成 Prompt

## 复制这个 prompt 到通义万相：

```
一个可爱友好的 AI 助手头像，机器人形象，科技感但温暖，
蓝白色调，简约现代风格，圆形头像，高清，专业设计，
3D 渲染，C4D 风格，柔和光线，白色背景
```

```

### 风格 4：可爱卡通风

蓝白配色，卡通风格，温暖亲切，3D 渲染
```

## 生成步骤

1. 打开通义万相：https://tongyi.aliyun.com/wanxiang
2. 点击"立即体验"或"体验 万相 2.6"
3. 登录阿里云账号
4. 选择"图像生成"功能
5. 粘贴上面的 prompt
6. 选择比例：1:1（正方形）
7. 点击生成
8. 下载喜欢的图片

## 参数建议

- **尺寸**: 1024x1024 或 512x512
- **比例**: 1:1
- **风格**: 3D 渲染 / C4D / 简约
- **数量**: 生成 4 张选最好的

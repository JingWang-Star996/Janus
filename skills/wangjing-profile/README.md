## 🚀 快速开始

### 方式 1：直接创作
```
帮我写个朋友圈，主题是 XXX
```

### 方式 2：基于对话
```
基于我今天的对话，写个总结发朋友圈
```

### 方式 3：审查文案
```
审查这个文案是否符合我的风格：[文案内容]
```

### 方式 4：更新画像
```
记住这个：我对 XXX 的看法是...
```

## 📊 画像维度

| 维度 | 内容 |
|------|------|
| 基本信息 | 姓名、职业、团队、行业 |
| 能力标签 | 技术、管理、商业、学习 |
| 做事风格 | 决策、沟通、问题解决 |
| 价值观 | 技术、原创、合作、风险 |
| 语言风格 | 词汇、句式、语气、禁忌 |
| 社交场景 | 朋友圈、小红书、演讲、私聊 |

## 🔄 更新机制

**触发条件**：
- 用户明确要求"记住这个"
- 用户纠正风格（"这个不像我"）
- 用户展现新特质/偏好
- 用户强烈反应（喜欢/讨厌）

**流程**：
```
检测新认知 → 提取关键信息 → 归类 → 写入 profile.md → 通知用户
```

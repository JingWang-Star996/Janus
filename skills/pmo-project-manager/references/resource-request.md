# 资源申请流程

## Unity 插件采购

### 流程

```
确认插件信息
    ↓
填写采购申请
    ↓
提交审批流程
    ↓
等待审批通过
    ↓
完成采购
    ↓
通知使用
```

### 所需材料

- 插件名称和链接
- 插件价格（美元）
- 使用用途说明
- 项目名称
- 申请人信息

### 时间预估

| 阶段 | 时间 |
|------|------|
| 准备材料 | 0.5 天 |
| 审批时间 | 1-3 天 |
| 采购时间 | 1 天 |
| **总计** | **2-5 天** |

### 注意事项

- ⚠️ 确认插件确实需要，避免浪费
- ⚠️ 价格较高的插件需要更充分的理由
- ⚠️ 采购前确认是否有免费替代方案

## 软件授权管理

### Unity Enterprise 席位分配

**流程**：
1. 统计需求人数
2. 提交申请
3. 分配席位
4. 记录使用信息

### 软件续费处理

**提前量**：到期前 2 周开始处理

**流程**：
1. 检查到期时间
2. 确认续费需求
3. 提交续费申请
4. 完成支付

## 美术资源申请

### 提前量参考

| 资源类型 | 提前量 | 备注 |
|----------|--------|------|
| 3D 角色 | 1-2 个月 | 制作周期长 |
| 场景资源 | 1-2 个月 | 需考虑修改 |
| 音频需求 | 2-3 周 | 排期紧张 |
| UA 素材 | 1 个月 | 多语言翻译 |

### 年底特别提示

⚠️ **年底期间资源尤其紧张**，建议提前 2-3 个月提交需求

### 申请流程

```
了解需求背景
    ↓
确认需求细节
    ↓
联系美术中心
    ↓
评估工作量
    ↓
确认排期
    ↓
提交正式申请
    ↓
跟进制作进度
    ↓
验收交付
```

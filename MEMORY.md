- ✅ 能合并的工具调用，不分开
- ✅ 能跳过的步骤，不执行
- ❌ 避免" mental notes"（记在脑子里）→ 写文件
- ❌ 避免重复已知上下文
- ❌ 避免过度解释

**如何执行**：
```python

    更新 status.json
    标记子任务完成状态
    更新进度百分比
    修改当前任务描述
```

## 🛠️ 技术笔记

### OpenClaw 限制

1. **sessions send 命令不存在**
   - `openclaw sessions` 只有 `list` 和 `cleanup`
   - 无法从外部脚本发送消息到指定会话

2. **子代理会话限制**
   - webchat 渠道不支持 `thread=true`
   - 子代理会话 ID 不被 `openclaw agent --session-id` 接受

3. **本地模式需要 API key**
   - `openclaw agent --local` 需要配置模型 provider
   - 当前使用阿里云百炼（bailian/qwen3.5-plus）

4. **WSL2 浏览器问题**
   - 缺少中文字体 → 显示方框
   - 解决方案：用 Windows 本机 Chrome

5. **浏览器工具限制**
   - 只能查看页面，无法点击/登录
   - Playwright 不可用

### 最佳实践

1. **文件队列方案**（最可靠）
   - 网页 → 写入文件 → 主会话查看 → 人工回复
   - 简单、可靠、100% 可用

2. **直接扮演**（最简单）
   - 在当前会话用王鲸身份回复
   - 无需复杂集成

3. **本机 Chrome**
   - WSL2 浏览器字体问题 → 用 Windows Chrome
   - 访问 `file:///home/z3129119/.openclaw/workspace/xxx.html`

4. **状态监控**（最新）
   - 使用 status.json 记录任务状态
   - 网页每 5 秒轮询更新
   - AI 需要在任务变化时立即更新

### 飞书表格公式技巧（2026-03-17）⭐

**来源**：实际操作经验总结

**创建技能**：`feishu-sheet-formula`

**核心要点**：
1. **跨表引用**：用 `IMPORTRANGE("URL", "范围")`
2. **类型匹配**：VLOOKUP时用 `VALUE()` 转换字符串为数字
3. **组合字段**：用 `SPLIT + MAP + TEXTJOIN` 处理多值
4. **错误处理**：用 `IFERROR()` 兜底

**典型公式**：
```
=IFERROR(TEXTJOIN("|",TRUE,MAP(SPLIT(B5,"|"),LAMBDA(x,VLOOKUP(VALUE(INDEX(SPLIT(x,"*"),2)),IMPORTRANGE("URL","A:D"),2,FALSE)))),"")
```
